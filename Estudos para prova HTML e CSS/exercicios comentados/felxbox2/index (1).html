🏷️ Introdução ao HTML

O que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação usada para estruturar páginas web.

Não é uma linguagem de programação — não executa lógica. Utiliza tags (etiquetas) para definir elementos na página.

HTML vs XHTML

Característica HTML XHTML
Sintaxe Flexível Estrita
Fechamento de tags Opcional (<br>, <img>) Obrigatório (<br />)
Letras das tags Maiúscula ou minúscula Sempre minúscula
Valores de atributos Podem ser omitidos Sempre atribuídos
DOCTYPE Opcional Obrigatório

Estrutura básica de um documento

Toda página HTML deve seguir esta estrutura:


<!DOCTYPE html>
<html lang="pt-BR">
  <head>
    <meta charset="UTF-8">
    <title>Título da Página</title>
  </head>
  <body>
    <h1>Olá, mundo!</h1>
  </body>
</html>

🎨 Introdução ao CSS

O que é CSS?

CSS (Cascading Style Sheets) define a aparência e o layout das páginas HTML. Separa o conteúdo (HTML) da apresentação (estilo).

Formas de usar CSS

  1. Inline: <p style="color: blue;"> — direto na tag. ❌ Evitar.
  2. Interno: <style> dentro do <head>. ⚠️ Uso limitado.
  3. Externo: arquivo .css separado + <link>. ✅ Recomendado.

Cascateamento e Herança

O CSS segue uma ordem de prioridade:

  1. !important → sobrescreve tudo (use com cautela)
  2. Inline → style="" na tag
  3. ID → especificidade 100
  4. Classe / pseudoclasse / atributo → especificidade 10
  5. Tag / pseudoelemento → especificidade 1
  6. Padrão do navegador

A última regra declarada tem mais peso se a especificidade for igual.

Propriedades herdáveis: color, font-family, font-size, visibility...

📦 Box Model

Todo elemento HTML é uma caixa retangular composta por 4 camadas (de dentro para fora):

  1. Content → o conteúdo real (texto, imagem)
  2. Padding → espaço INTERNO entre conteúdo e borda
  3. Border → a borda ao redor do padding
  4. Margin → espaço EXTERNO que separa de outros elementos

box-sizing: border-box

Por padrão (content-box), width conta só o conteúdo — padding e border são somados por fora. Com border-box, padding e border ficam dentro da largura declarada.
